RentWheels is a full-stack, responsive platform designed for a seamless car rental experience. Users can browse, book, and manage listings through an interactive and secure interface.

**🔗 Live Demo:** [https://rentwheels-car.web.app](https://rentwheels-car.web.app)

---

## ✨ Key Features

- **Smart Browsing:** Explore a wide range of vehicles with real-time availability, search by name, and category-based filtering.
- **Dynamic Booking System:** Intuitive car details page with automated cost calculations and instant booking options.
- **Secure Authentication:** Robust user login and registration powered by **Firebase Authentication**.
- **User Dashboard:** Dedicated sections for **My Bookings** and **My Listings** to manage personal car uploads and rentals.
- **Premium UI/UX:** Built with **Tailwind CSS** and **DaisyUI**, featuring smooth animations with **Framer Motion**.
- **Robust Backend:** Secure APIs developed using **Node.js** and **Express.js**, integrated with **MongoDB** for reliable data management.
- **Interactive Elements:** Uses **React Tooltip** for pricing details and **SweetAlert2** for beautiful notifications.
